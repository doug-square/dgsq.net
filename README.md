# dgsq's site

This is the code for dgsq.net. Using layout and code originally developed at (https://lunahd.neocities.org/)[https://lunahd.neocities.org/].

To deploy, set up the gh-pages branch within the main repo folder like so:
```
git worktree add public gh-pages
```

Then the generated files will end up in that folder and can be pushed to
that branch when the site needs to be updated.
